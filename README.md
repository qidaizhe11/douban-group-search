# douban-group-search

> 豆瓣小组用户搜索/分析工具

## 功能

- 搜索小组成员（按城市过滤）

- 遍历小组最近发帖（按城市/性别/昵称等过滤）

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# run unit & end-to-end tests
npm test

# lint all JS/Vue component files in `src/`
npm run lint

```
